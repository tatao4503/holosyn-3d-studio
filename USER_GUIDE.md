# HOLOSYN 완전 사용설명서

> 3D 시제품 공간 발표 스튜디오. 만든 사람도 까먹는 버전 — 현재 기준 전체 기능 정리.

---

## 🟢 비기너 3단계 (이것만 알면 됨)

1. **부팅** — 페이지 열면 모달 → `HOLOSYN 엔진 기동` 클릭 (브라우저 오디오 정책상 클릭 1회 필수)
2. **① 넣기** — 왼쪽 `샘플 모델로 시도하기`에서 **ST,AND / Drone / Ring / Car / Core / Exo-Suit** 중 하나 클릭
   - 또는 위쪽 영역에 내 `.glb / .gltf / .obj` 파일이나 이미지를 드래그
3. **② 보기** — `HOLO`(구조) / `COLOR`(실제 색) / `PART`(선택 부품) 중 하나 선택
4. **③ 발표·공유** — 구조·부품·쇼케이스·공유 URL까지 30초 흐름으로 자동 진행

여기까지가 95%. 나머지는 필요할 때만.

---

## 🧭 비기너 / 프로

- **비기너** — `① 넣기 → ② 보기 → ③ 발표·공유`만 표시
- **프로** — 분해·조립, 진단, 타임라인, 측정, 상세 내보내기까지 표시

## 🎛️ 왼쪽 패널 (제작·연출)

| 섹션 | 하는 일 |
|---|---|
| **3D 모델 & 이미지 임포트** | 내 파일 드롭 / 컴퓨터에서 열기 |
| **샘플 모델로 시도하기** | 6개 프리셋 (ST,AND가 맨 앞) |
| **입체 분해도 (Exploded)** | 슬라이더로 부품 분리. 0% / 50% / 100% 스냅 버튼 |
| **로봇 조립 시퀀스** | STEP 1~4 단계별 조립/분해 연출 |
| **ST,AND 시연 컨트롤** *(ST,AND 선택 시만)* | **펼침/접힘** 토글 + **BASIC/CLEAR/CUSTOM** 라인업 |
| **AI 진단 / 시제품 점검** | 자동 점검 리포트 |
| **QUICK STYLES** | 미니멀/택티컬/매트릭스 분위기 |
| **HOLOGRAM TUNING** *(프로)* | Glow/Rotation/Scanline/Glitch/Scale 슬라이더 |
| **WAVE WAVELENGTH** | 색상 팔레트 |
| **STUDIO SETTINGS** *(프로)* | Sound / Auto Rotate / Spatial / Pyramid 스위치 |

## 📐 뷰포트 위 도구 (화면 안 상단)

- **3D 스케치** — 화면에 자유 드로잉 (발표 중 강조)
- **3D 측정** — 모델 표면 점 A→B 클릭하면 거리(mm) 표시. *ST,AND는 실제 치수(앞판=150mm)로 보정됨*
- **더블클릭** — 모델 표면에 콜아웃 핀(주석) 꽂기

| **실시간 투사 텔레메트리** | 자동 표시 스펙 |
| **EXPORT SUITE** | 아래 ↓ |

### 내보내기 (EXPORT SUITE + 하단)
- **GLB 다운로드** — 3D 모델 파일
- **사양 JSON** — 스펙 데이터
- **PNG 캡처** — 현재 화면 (사업계획서·SNS용)
- **클라이언트 브리프** — 현재 상태로 발표용 한장 문서(마크다운) 자동 생성
- **데모 팩 / 핸드오프 팩** — 시연 설정 번들
- **PORTABLE PROJECT** — 실제 모델과 발표 상태를 `.holosyn` 한 파일로 저장·복원

## 🔮 Pepper's Ghost 실물 시연 (전시·부스)

1. 프로 모드 → STUDIO SETTINGS → **PYRAMID REFLECTION** ON
2. 화면 4분할 → 아크릴 피라미드를 화면 위에 뒤집어 올림
3. 실내 조명 끄면 공중에 입체 홀로그램

---

## 📦 다른 컴퓨터로 프로젝트 옮기기

1. 우측 `PROJECT SNAPSHOT` 아래 `번들 저장` 클릭
2. 생성된 `.holosyn` 파일을 전달
3. 받는 컴퓨터에서 엔진 기동 후 `번들 열기`

모델 GLB, 카메라, 조명, 타임라인, 발표 노트와 저장된 치수가 함께 복원됩니다. 모델 데이터는 최대 80MB까지 번들에 포함되며, 열 때 SHA-256 무결성을 검사합니다.

---

## 🎯 ST,AND 피칭 추천 플로우

1. ST,AND 선택 → 자립 명패 등장
2. **펼침↔접힘** 토글로 "접어서 보관, 펴서 자립" 시연
3. **PART SCAN** ◀▶ 으로 앞판→리빙힌지→받침대 하나씩 설명
4. **3D 측정**으로 "앞판 150mm" 실측 강조
5. **BASIC/CLEAR/CUSTOM** 라인업 비교
6. **클라이언트 브리프 / PNG** 내보내 사업계획서·심사 핸드아웃

`30s PITCH`는 홀로그램 구조 → HYBRID 부품 포커스 → PRODUCT 컬러 공개 순서로 자동 연출됩니다.

## ❓ 막힐 때

- **버튼이 안 보임** → 프로 모드인지 확인 (일부는 프로 전용)
- **단일 메쉬 GLB라 분해 안 됨** → 파트 분리된 모델 필요 (안내 토스트 뜸)
- **소리 안 남** → 부팅 클릭 1회 필요 / `M`으로 토글
- **처음부터 다시** → 새로고침
