# HOLOSYN Demo Script

Use this when showing the prototype to another person.

## Private Rehearsal

Use this when you want to check the product without showing it to anyone yet.

1. Boot the engine and pick one sample.
2. Follow `NEXT ACTION` once.
3. Turn on `Part Scan` and jump through two components with the Part Map rail.
4. Apply one Demo Scene Preset.
5. Save a Project Snapshot.
6. Check Final Readiness and stop when it reaches `DEMO READY` or better.

## 30-Second Version

1. Open HOLOSYN and boot the engine.
2. Say: "This is a local 3D prototype presentation viewer."
3. Pick a sample prototype and point out the import quality card.
4. Click the card's `NEXT ACTION` button to jump into the safest next presentation step.
5. Use a quick viewport swipe or modifier-wheel/pinch gesture to show the Gesture Pilot response.
6. Click `SHOWCASE` for the clean presentation view.
7. Run `FINAL PASS`, then point to the Showtime Pass and Beta Preflight checks.
8. Click `시연 패키지 생성` and explain that it bundles the brief, snapshot, and handoff manifest.

## 3-Minute Version

1. Boot
   - "The app runs locally and turns a prototype file into a presentation-ready 3D view."

2. Model
   - Pick Drone, Ring, Car, Core, or Exo Suit.
   - Or drop in a `.glb`, `.gltf`, `.obj`, or image file.
   - For multi-part GLB/OBJ files, point out that HOLOSYN auto-maps imported mesh parts into Part Scan.
   - Point to the import quality card: it tells you whether the model is presentation-ready or limited, then recommends the next action.

3. Structure
   - Use the exploded view.
   - Use Gesture Pilot for a more tactile spin or explode adjustment.
   - Turn on `Part Scan`, then use the Part Map rail to jump directly between components.
   - Double-click a part label to rename it.
   - Use `라벨 초기화` if you want to restore defaults.

4. Present
   - Click a Demo Scene Preset for a ready-made camera, render, environment, and timeline setup.
   - Use `Suit Lab` when you want a powered exosuit concept experiment.
   - Save a Project Snapshot so the same setup can be restored later.
   - Click `데모 런` if you want the simple guided flow instead.
   - Open `Timeline` to stage or replay a keyframe presentation.
   - Use Showcase mode for a cleaner audience-facing view with automatic Part Scan focus.

5. Pro tools
   - Use AI Assistant only when a local Gemini key is available.
   - Use Collaboration only when real-time link sharing is part of the demo.

6. Handoff
   - Review Beta Preflight before exporting.
   - Point to the Handoff Pack Part Map readiness badge before generating deliverables.
   - Check Final Readiness and aim for Demo Ready or Handoff Ready.
   - Click a Handoff checklist chip if you need to jump back to model, demo, timeline, or export prep.
   - Click the Handoff Pack next-step guidance if any item is still pending.
   - Leave HQ BOOST on before final PNG/spec-card export when the machine can handle it.
   - Run FINAL PASS to lock HQ Boost, save the current snapshot, and focus the export path.
   - Click `시연 패키지 생성` for the one-click Demo Pack.
   - Export PNG for a presentation image.
   - Export JSON for specs.
   - Export GLB for sharing the model asset.
   - Export Client Brief Markdown when you want a human-readable summary with Part Map talking points.
   - Export Handoff Manifest for the demo setup, quality state, Part Map summary, and recommended deliverables.

## Best Demo Line

"It is not trying to replace CAD. It turns a CAD or product concept into something you can explain in a meeting."
